# Sylphrena - a diary (WIP)

An online Diary. Tell your hopes, dreams, and desires to your twin soul.

## Instructions

1.  Clone this repo
2.  Run `npm install`
3.  Run `npm start`, **localhost:3000** will open up in your default browser
